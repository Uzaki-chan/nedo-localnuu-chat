скачайть файлы из репозитория
установите node js c офицального сайта https://nodejs.org/en
откройте терминал и перейдите в папку, где находится скачанные файлы
выполните команду 'npm init -y'
затем выполните команду 'npm install express body-parser'
запустите сервер для этого выполните команду 'node server.js'
после запуска сервер будет доступен по порту http://localhost:3000/
